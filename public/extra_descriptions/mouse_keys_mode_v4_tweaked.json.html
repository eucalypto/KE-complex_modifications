Mouse keys Mode v4 (tweaked from @tekezo rev 3)

You can operate mouse via keyboard. (motion, click and scroll)
Mouse keys mode can be activated by pressing by the following procedure pressing left_command, left_option, left_control, m
To leave mouse keys mode, you can press left_command, left_option, left_control, m again or escape

Optionally you can also use caps_lock to enter or leave the mouse keys mode

Key bindings:
  • h: mouse left
  • j: mouse down
  • k: mouse up
  • l: mouse right
  • v: left click
  • b: middle click
  • n: right click
  • f: fast mode (fast mouse move by f+hjkl)
  • d: slow mode (slow mouse move by d+hjkl)
  • s: scroll mode (scroll by s+hjkl)
  • u: set mouse cursor position to the center of current display
  • i: set mouse cursor position to the center of main display
  • o: set mouse cursor position to the center of second display
  • p: set mouse cursor position to the center of third display
Optional key bindings (if enabled):
  • left_arrow: mouse left
  • down_arrow: mouse down
  • up_arrow: mouse up
  • right_arrow: mouse right
  • shift: fast mode (fast mouse move by shift+[arrow])
  • option: slow mode (slow mouse move by option+[arrow])
  • control: scroll mode (scroll by control+[arrow])

Note that if optional keybinding is used, you can still exit the mouse keys mode using escape.
This is due to conflict for option and control used for mouse speed adjustment
