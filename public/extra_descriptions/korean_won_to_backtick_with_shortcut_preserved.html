Korean Won (₩) to Backtick (`) with shortcut preserved

MacOS의 단축키 사용의 영향 없이, ₩ 키를 ` 키로 변경합니다.
오직 단일로 ₩ 키를 눌렀을 때만, ` ( + ⌥ left_option ) 키로 변경됩니다.
다른 키와 조합하여 ₩ 키를 눌렀을 때는, ₩ 키로 남아 단축키에 영향을 주지 않습니다.

  • ₩ -> ` ( + ⌥ left_option )
  • ⌘ command + ₩ -> ⌘ command + ` ( + ⌥ left_option )
  • Other keys + ₩ -> Other keys + `
